Parse Html Doc
================================

1. Opis 
******************************** 

Celem projektu "Parse Html Doc" jest wyłuskanie i wyświetlenie z dokumentu "strona.html" (strona html) specyficznych informacji (email, nazwa firmy, adres).


2. Kompilacja 
******************************** 

Należy kompilować kompilatorem w wersji g++-4.9
